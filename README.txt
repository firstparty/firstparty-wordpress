Automatically add Firstparty's analytics script to your WordPress site.

== Description ==

Enter a Source Write Key below and Firstparty will automatically collect Page Viewed Events for all of your theme pages.

WP Admin pageviews will not be collected.

After installing, you may also access the window.firstparty object in your frontend javascript, which allows you to
collect custom Events like form completions and button clicks.


== Installation ==
1. Upload `firstparty.zip` via the plugins menu in your WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Source Write Key and Domain in the Firstparty settings page.
